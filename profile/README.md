# Signalbash

Signalbash is a free, open source audio plugin and website for tracking active time spent working in DAWs and other audio hosts.
The plugin sends data to your Signalbash account to display your activity over time, and allow you to compare it against others on the global leaderboard.

## Links

- [Signalbash Audio Plugin](https://signalbash.com/plugin-download)
- [Live DAW Activity Leaderboard](https://signalbash.com/)
- [Support Site](https://info.signalbash.com/)
