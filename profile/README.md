# Signalbash

Signalbash is a free, open source audio plugin and website for tracking active time spent in DAWs and other audio hosts. The plugin reports activity to your Signalbash account, where you can review profile stats, streaks, and leaderboard activity.

## Links

- [Download DAW Plugin](https://signalbash.com/plugin-download)
- [Live DAW Activity Leaderboard](https://signalbash.com/)
